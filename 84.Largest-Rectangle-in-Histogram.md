#### 3. 代码

###### 3.1 Brute Force

```java
// LTE
class Solution {
    public int largestRectangleArea(int[] heights) {
        int len = heights.length;
        int maxArea = 0;
        for (int i = 0; i < len; i++) {
            int minHeight = heights[i];
            for (int j = i; j < len; j++) {
                if (minHeight > heights[j]) {
                    minHeight = heights[j];
                }
                if (maxArea < (j - i + 1) * minHeight) {
                    maxArea = (j - i + 1) * minHeight;
                }
            }
        }
        return maxArea;
    }
}
```
